# RoleLogic v1.9.0

*Released April 1, 2026*

---

## What's New

### Dashboard Welcome Screen

Opening your server's dashboard now greets you with a clean "Get Started" overview. It shows a simple three-step path and two quick-action cards — one to connect an external platform and one to create a role condition. If your plugin quota is full, the platform card is automatically grayed out so you always know where you stand before you click.

### Featured Plugins on the Homepage

The RoleLogic homepage now highlights popular external platforms — YouTube, Twitch, Top.gg, GitHub, Genshin Impact, and more — so new users can immediately see what integrations are available and jump straight into setup.

### Plugin Installation Count on Status Page

The Status page now shows how many servers have each plugin installed, giving you a quick sense of how widely used each integration is across the community.

### Quota Usage Bars in the Sidebar

Your role-condition and plugin usage limits are now displayed as compact progress bars right in the sidebar. When you're close to or at a limit, the bar turns red so you know at a glance without having to visit the Upgrade page.

### Smarter Navigation After Deleting a Rule

Previously, deleting a rule would drop you on a blank page. Now RoleLogic automatically takes you to your most recent remaining rule — or opens the new-rule form if no rules are left — so you can keep working without interruption.

### Deep-Link Directly to a Plugin Setup

Sharing a link to a specific integration is now possible. A URL with the right parameters will open the dashboard and pre-select the chosen plugin, skipping the marketplace browse step entirely.

---

## Improvements

- **Performance** — System health and plugin status checks are now cached for 15 seconds, reducing server load during high traffic without affecting the accuracy of what you see.
- **SEO & page titles** — Every dashboard page now has a proper browser tab title and meta description, making bookmarks and browser history more descriptive.
- **Refund policy** — The refundable window shown in checkout and policy pages has been corrected to **14 days** across all supported languages.
- **Twitch icon** — The Twitch platform icon has been refreshed with an updated visual.

---

## Bug Fixes

- Fixed an issue where the Plugin Marketplace overlay could appear behind other interface elements.
- Fixed a crash that could occur when deleting the last rule in a server with no remaining rules.
- Fixed an edge case where an empty role-links list would not display the "Add your first link" prompt correctly when a new link was being created at the same time.
