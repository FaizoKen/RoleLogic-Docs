# RoleLogic v1.11.0

**Release Date:** May 8, 2026

---

## What's New

### More Generous Free Plan — Now 5 Rules and 5 Role Links Per Server

The free plan just got a lot roomier. Every server can now create up to:

- **5 Role Conditions** (up from 2)
- **5 Role Links** (up from 2)

That's more than double the headroom on the free tier — enough to set up a real automation flow without needing to upgrade. Existing servers get the new limits automatically; nothing to configure.

### Faster Real-Time Role Conditions

Role Conditions now react about **twice as fast** when a member's roles change. Previously the bot would briefly wait (around 10 seconds) before evaluating the rule to avoid reacting to rapid-fire role updates; that wait time has been cut roughly in half. Members will see their roles update sooner after qualifying for (or losing eligibility for) a role.

### Three New Platform Integrations

RoleLogic can now hand out Discord roles based on activity from three more platforms:

- **TikTok Creator Role.** Verify a member's TikTok account and grant roles based on their public creator stats — verified-creator status, follower count, video count, and more. Great for creator communities that want to reward and recognize active TikTokers.
- **Roblox Player Role.** Verify a member's Roblox account and grant roles based on group memberships, group rank, badges, gamepasses, verified-badge status, or in-game stats reported by Roblox game owners.
- **Loot Labs Reward Role.** Reward members for completing Loot Labs offers with automatic Discord role grants.

Each new integration has its own dedicated landing page at **rolelogic.faizo.net/integrations**, with full setup guides, FAQs, and clear privacy details for the data each platform shares with us.

### "Trusted Worldwide" Showcase on the Homepage

The homepage now features a new **Trusted Worldwide** section that highlights real Discord communities using RoleLogic — including Discord Partners and Verified Servers — with their member counts and badges. It's a quick, at-a-glance way for new visitors to see the kinds of communities that already rely on RoleLogic.

### Browse Plugins, Right From an Empty Role Link

Setting up your first Role Link is now much more discoverable. When you open the Role Links page on a fresh server and there's nothing to show yet, you'll see a prominent **"Add Plugin"** button that drops you straight into the plugin marketplace — no hunting required.

## Improvements

### Refreshed Sidebar Design

The dashboard sidebar has been polished for clarity:

- Each section now has its own subtle accent color — Get Started, Activity Log, Role Links, Linked Guilds, Add New Rule, and Test All Rules are easier to tell apart at a glance.
- The active page is highlighted with a clean colored accent bar instead of a heavy background.
- Counts (Role Links, Linked Guilds) now appear as compact pill badges that are easier to read.
- Spacing, icons, and hover states have been tightened across the entire navigation.

### Confirmation Before Purging Linked Servers

The **Purge** button on the Linked Guilds page now opens a clear confirmation dialog before removing all action roles assigned by that linked server — protecting you from accidental clicks on a destructive action.

### Cleaner Plugin Editor Workflow

When you save changes to a plugin's configuration, the editor now closes automatically and returns you to your Role Link list. No more visually re-loading the same panel after a successful save.

### Auto-Refresh After Fixing Bot Permissions

If you switch to another tab to fix the bot's role placement (or any other permission issue) and then come back, the **Fix Role Guide** screen will now refresh automatically — so you don't have to manually reload the page to see that the issue is resolved.

### Smarter Homepage Sign-In Button

The homepage's main call-to-action button now waits gracefully if your sign-in status hasn't been determined yet — showing a small loading indicator instead of doing nothing on the first click. Once your session is ready, it sends you to the right place automatically.

### Page Titles for Easier Tab Switching

Browser tabs and the page title now reflect what you're looking at — for example, your server name on the dashboard, "New Rule" when creating a rule, or "Edit Rule" when editing one. Easier to find the right RoleLogic tab when you have several open.

### Plugin Marketplace Polish

- The marketplace category pills and plugin cards have a slightly cleaner layout.
- The integrations link in the site header now points to a dedicated, faster integrations hub instead of an external page.

## Bug Fixes

- **More resilient server search on the dashboard.** When loading the server picker, RoleLogic now applies a sensible timeout on its lookup to Discord and logs clearly when something goes wrong instead of silently falling back, so partial outages on Discord's side won't cause the dashboard to hang.
- **Fewer timeouts when listing many servers.** Increased the internal time budget for listing servers, so users with large numbers of servers no longer see "request timed out" errors during dashboard load.
- **Verified communities show first in the showcase.** The "Trusted Worldwide" list now consistently puts Verified and Partnered servers ahead of others, regardless of size.
- **Plugin URL handling.** Trailing slashes in plugin URLs are handled correctly when matching role links to their plugin.
- **Plugin marketplace modal layout.** Fixed a small visual inconsistency where the modal's rounded corners and category pill spacing didn't match on certain screen sizes.

## Under the Hood

A round of behind-the-scenes work focused on making rolelogic.faizo.net itself faster, friendlier to search engines, and easier for users to discover:

- **Static landing pages for every integration.** Each plugin now has its own pre-rendered, fully-indexable landing page at `/integrations/<plugin>/`, so users searching for things like "Twitch follower Discord role" or "Roblox Discord verification" can find RoleLogic directly.
- **Faster page loads.** The web server now compresses HTML, JavaScript, CSS, and JSON responses on the fly, noticeably reducing load times on slower connections.
- **Cleaner search engine indexing.** A new set of redirect rules and structured-data tags resolve a long-standing "Redirect error" reported in Google Search Console, so RoleLogic's pages are picked up reliably and consistently.
- **Refreshed legal documentation.** The Privacy Policy and Terms now include dedicated, clearly written sections explaining exactly what data RoleLogic accesses (and doesn't access) when you use the Roblox or TikTok integrations — and how to revoke access at any time.

The end result: a faster homepage, easier-to-find integration pages, and a more confident, transparent experience for new and existing users alike.
