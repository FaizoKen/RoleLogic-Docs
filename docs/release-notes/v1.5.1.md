# RoleLogic v1.5.1

_Released January 25, 2026_

---

## What's New & Improved

- **Enhanced rule change logs** — Activity logs now show more detailed information for rule updates.
- **New community documentation released**
  - Website: [docs-rolelogic.faizo.net](https://docs-rolelogic.faizo.net)
  - GitHub: [github.com/FaizoKen/rolelogic-docs](https://github.com/FaizoKen/rolelogic-docs)
- **Added LLM documentation** — Improves AI discovery and documentation indexing. You can now ask ChatGPT or other AI tools with web search support for RoleLogic guides.
- **Webhook log UI improvements** — Edit or delete webhook logs directly from the list, without needing to select them.
- **Fixed rare rendering issues** in the Test and Rule sections affecting older web browsers.
- **Added retry backoff for Discord API user verification** — Reduces errors across all user-related API calls.
- **Production infrastructure improvements** — Added cross-server database backup sync. One server down, data still safe.
- **Minor sidebar UI improvements** for better usability and clarity.
- **Cleaned up role icons UI** — Current guild roles no longer show the guild icon. Only cross-server roles display an icon, keeping the list cleaner since most roles are from the current guild.
