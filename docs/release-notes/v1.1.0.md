# RoleLogic v1.1.0

*Released September 25, 2025*

---

## New Features

- **Manual Live Controls:** You can now manually start and pause the live process using dedicated buttons. This gives you direct control without needing to wait for the automatic start.

---

## Enhancements

- **Faster Rule Propagation:** The rule cache interval on our worker has been reduced from 3 minutes to 1 minute. This ensures that any changes to your rules are applied significantly faster.
- **General Code Refinements:** Implemented several minor code improvements for better performance and stability.

---

## Bug Fixes

- **Corrected Discord Link:** Fixed a broken hyperlink to the Discord server that appeared when the status was "Live".
