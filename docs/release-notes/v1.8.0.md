# RoleLogic v1.8.0

*Released March 26, 2026*

---

## What's New

- **System Status Page** — You can now check if RoleLogic and its services are running smoothly, right from the website. A new "Status" link in the footer takes you to a live dashboard showing the health of the bot, database, and all connected plugins.

- **Log Conditions for Role Links** — Role link plugins can now use log conditions. Choose whether to log when members are added, removed, or both — giving you more control over what shows up in your activity logs.

- **Plugin Marketplace Improvements** — The role link setup now includes a built-in plugin picker, making it easier to browse and select plugins from the marketplace. You'll also see plugin logos and a link to suggest new plugins on our Discord.

- **Smarter Role Link Dashboard** — The role links page (now called "Role Link Plugins") has been redesigned with search, filtering (all / enabled / disabled), and sorting (by name, users, plugin, or recently updated) so you can manage large setups more easily. Helpful hints guide you through creating a new role link.

- **YouTube Subscriber Role Plugin** — A new plugin lets you assign Discord roles based on YouTube channel subscriptions. Link your YouTube account, and the bot automatically gives or removes a role depending on your subscription status.

---

## Improvements

- Dropdowns and selectors across the dashboard now look and feel more polished, especially on mobile devices.
- Log selectors no longer stretch awkwardly on smaller screens.
- The plugin list now loads faster thanks to smarter caching.
- Network requests to plugins are now more reliable with automatic retries.
- Fixed an issue where the rule editor could show an error in certain conditions.
- Fixed a validation bug when creating role links.
- Fixed the debounce issue that could cause unnecessary repeated updates.

---

## Privacy & Policy

- Updated Privacy Policy (effective March 26, 2026) now includes a dedicated section on how YouTube and Google API data is handled, in full compliance with Google's Limited Use requirements. You can review it anytime from the footer.
