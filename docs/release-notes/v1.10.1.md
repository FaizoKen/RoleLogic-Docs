# RoleLogic v1.10.1

**Release Date:** April 5, 2026

---

## What's New

### Mention Controls in Log Messages

You can now choose whether user mentions and role mentions in your log messages actually send notifications (pings) to the mentioned people or roles.

- Two new toggles have been added to the **Log Editor**: **"Allow user mentions to ping"** and **"Allow role mentions to ping"**.
- When turned off, mentions will still appear in the message as styled text, but they won't notify anyone — perfect for keeping logs clean and quiet.
- When turned on, mentioned users or roles will receive a notification as usual.
- By default, new log messages have mentions turned on so your existing setup continues to work as expected.

This feature is fully available in all supported languages: English, Arabic, German, Spanish, French, Malay, and Portuguese (Brazil).

---

## Improvements

### Increased Role Link User Limit for Basic Plan

Servers on the **Basic plan** can now apply role links to up to **1,000 users**, up from the previous limit of 100. This means larger communities on the Basic plan can benefit from automatic role assignments without needing to upgrade.

---

## Bug Fixes

- Fixed an issue where the dashboard could fail to load for users who don't have any servers they can manage. The page now loads correctly and shows an empty server list instead of getting stuck.
