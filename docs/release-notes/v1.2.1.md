# RoleLogic v1.2.1

_Released November 2, 2025_

---

## What's New & Improved

- Added a **255-character limit** for log names to prevent excessively long entries.
- Fixed an issue where **long log names** caused the interface layout to break.
- Refined the **log editor layout and spacing** for better readability and alignment.
- Added a **note reminding users** that upgrading removes the watermark from log messages.
- Fixed a **bug with the Undo feature**, which previously caused errors or affected other guilds when using web undo.
- Updated **role logging behavior** — logs now display **only roles relevant to their associated rule(s)**.
- Changed the bot's **Activity status** to show the **dashboard website**.
