# RoleLogic v1.1.2

_Released October 4, 2025_

---

## What's Changed

- **New Members Instant Role Assignment:** New members now trigger real-time rules immediately upon joining the guild — ensuring rules with the everyone role get their roles faster.

- **Bot War Detection:** RoleLogic can now detect "bot wars" — situations where other bots manage the same roles (e.g., RoleLogic adds Role A while another bot removes it repeatedly). When detected, RoleLogic will **automatically stop the guild rules** and **notify you via the Admin Dashboard**.

- **Expanded Match AND Conditions:** You can now add **up to 9 additional match AND conditions** (10 total, including the main one). Previously, the limit was 4 in total.

- **Improved Rule Editor UI:** AND conditions are now **collapsible**, with data indicators for better clarity — keeping your interface clean even when managing complex rules.
