# RoleLogic v1.10.0

**Release Date:** April 3, 2026

---

## What's New

### Smarter & More Reliable Role Verification

We made significant improvements to how RoleLogic checks your members' roles and server memberships — especially in larger servers or during periods of high activity.

- **No more wrong results from temporary errors.** Previously, if Discord had a brief hiccup while verifying whether a user belongs to a server, RoleLogic might silently treat them as a non-member. Now, the system automatically retries any failed checks before giving up — and if the problem persists, it reports an error clearly instead of acting on incomplete data. This ensures your role assignments are always based on accurate information.

- **RoleLogic now stays within Discord's request limits more gracefully.** A new smart request throttle has been added behind the scenes. When RoleLogic is processing many users at once, it now paces its requests to Discord to avoid hitting rate limits — leading to fewer slowdowns and more consistent performance during bulk operations.

---

## Improvements

### Dashboard Visual Refresh

- The **Dashboard Overview** cards for "Role Links" and "Role Conditions" now display a preview image, making it easier to understand what each feature does at a glance.

### Easier Navigation

- A new **"Get Started"** button has been added to the server sidebar. It takes you directly back to the server overview, making it simpler to navigate when setting up a new server.

---

## Bug Fixes

- Fixed an issue where bulk membership verification could return inaccurate results if any individual server lookup failed. The system now correctly surfaces an error rather than proceeding with incomplete data.

---

## Under the Hood

This release includes internal improvements to how RoleLogic communicates with Discord, making the bot more resilient and efficient when handling large numbers of users or servers simultaneously.
