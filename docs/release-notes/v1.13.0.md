# RoleLogic v1.13.0

**Release Date:** May 20, 2026

---

## What's New

### Premium Servers Get a Much Faster Background Sync

Alongside instant real-time processing, RoleLogic runs a periodic background sweep that re-checks every member to catch anything missed in real time — for example during a bot restart or a brief Discord outage. Until now that safety sweep ran at the same pace for every server. Premium servers now get a dramatically faster one.

- **Re-checked about every 2 minutes** instead of every 10.
- **Far more members scanned per pass**, so large servers fully catch up much sooner. A 100,000-member server now fully reconciles in roughly **10 minutes on premium** versus a few hours on free; very large servers (1M+ members) drop from over a day to under two hours.
- **Real-time processing is unchanged and identical on every plan** — when a member's roles change, rules still apply within seconds whether you're free or premium. This release only speeds up how quickly the periodic safety net closes any gap.

Free servers are unchanged: instant real-time processing plus the familiar ~10-minute background sweep — already plenty for the vast majority of servers.

See **[Plans & Pricing](../plans)** for the full free vs. premium comparison.

---

## Under the Hood

- **Tier-aware scheduler.** The background sweep now schedules each server on its own cadence based on plan, instead of one fixed interval for everyone. Premium servers come due more often and are scanned more deeply per pass.
- **Same safety guarantees.** Every Discord request still passes through a global rate-limit budget and a strict concurrency cap, so the faster premium cadence stays comfortably within Discord's limits. Under heavy load the system slows gracefully rather than ever pushing past a safe request rate — premium is faster, never riskier.
- **No configuration needed.** The faster sync applies automatically to servers with active premium quota; it turns off again if premium lapses.
