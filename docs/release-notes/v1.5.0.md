# RoleLogic v1.5.0

*Released January 16, 2026*

---

## New Feature: Activity Log

We've added an **Activity Log** to help you track all changes made to rules and settings in **your server RoleLogic dashboard**.

### What's included

- View a complete history of changes directly from **your dashboard**
- See who made changes and what was updated, all in one place
- Enable **real-time log notifications** by selecting a **Notification Channel** in the Activity Log dashboard

This update improves **transparency, admin control, and server safety**, making server management easier and more reliable.
