# RoleLogic v1.2.2

_Released November 21, 2025_

---

## Premium Balance

We've noticed that while RoleLogic has **over 300 active users**, only a few have opted for Premium. To make Premium more meaningful and help **support our hosting costs to keep the bot online**, we've made the following updates:

- **Non-upgraded guilds** now have **2 default rules** (down from 5). Basic usage is still fully supported!
- **Premium status** now requires **5 upgrades** (previously 1), reflecting active engagement.

---

## UI Improvements

- Optimized re-render logic for a smoother, less stressful experience.
- Minor visual refinements for a cleaner look.

These changes are designed to **make Premium more valuable** while keeping the bot enjoyable and reliable for everyone.
